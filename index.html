Tools Reference

12 MCP tools across three domains

search_pubmed Literature

Search PubMed with filters for date range, article type, and sort order. Returns PMIDs, titles, authors, journals, and DOIs.

📄 tools/search.ts → lib/ncbi.ts

get_abstract Literature

Full structured abstract — labeled sections (background, methods, results, conclusions) with keywords and MeSH terms.

📄 tools/abstract.ts → lib/ncbi.ts, xml-parser.ts

get_full_text Literature

Full text of open-access PMC articles. Parsed sections, figure/table captions, reference counts.

📄 tools/fulltext.ts → lib/ncbi.ts, xml-parser.ts

find_related Literature

Find similar articles using PubMed's neighbor algorithm, ranked by relevance score.

📄 tools/related.ts → lib/ncbi.ts

format_citation Literature

Generate formatted citations in APA, Vancouver, Harvard, or BibTeX style from a PMID.

📄 tools/cite.ts → lib/ncbi.ts, xml-parser.ts

trending_papers Literature

Recent papers on a topic, optionally filtered to high-impact journals (Nature, NEJM, Lancet, JAMA, etc.).

📄 tools/trending.ts → lib/ncbi.ts

get_uspi Drug Label

US Prescribing Information from DailyMed — indications, dosing, warnings, contraindications. Parsed from FDA SPL XML.

📄 tools/uspi.ts → lib/dailymed.ts, xml-parser.ts

get_smpc Drug Label

UK Summary of Product Characteristics from eMC — numbered SmPC sections via HTML scraping.

📄 tools/smpc.ts → lib/emc.ts

compare_labels Drug Label

Side-by-side US (USPI) vs UK (SmPC) comparison. Spot regulatory differences in indications, warnings, dosing.

📄 tools/compare-labels.ts → lib/dailymed.ts, emc.ts, label-mapping.ts

search_by_indication Drug Label

Find drugs approved for a medical condition. Searches FDA via OpenFDA, cross-references UK availability on eMC.

📄 tools/search-indication.ts → lib/openfda.ts, emc.ts

search_trials Trials

Search ClinicalTrials.gov — filter by condition, intervention, status, phase. Returns NCT IDs, sponsors, enrollment.

📄 tools/trials-search.ts → lib/clinicaltrials.ts

get_trial Trials

Full trial details — eligibility, study design, arms, primary/secondary outcomes, locations, associated PMIDs.

📄 tools/trials-detail.ts → lib/clinicaltrials.ts
